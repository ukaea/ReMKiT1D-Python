=========
Tutorials
=========

A set of tutorials is provided to cover most basic and some advanced concepts in ReMKiT1D. Users are encouraged to read the original paper as well as browse the full examples in addition to these tutorials.

The tutorials are provided as separate notebooks, so users are also encouraged to play around with them. Note that the are not meant to produce runnable examples, for which the users are directed towards the plethora of examples.
